jpFileUploader has a few requirements before you get started.

  1. You need jQuery. Being this is a jquery plugin that almost goes without saying but you never know.
    You can find jQuery here: https://developers.google.com/speed/libraries/#jquery
  2. This plugin is highly dependent on the style from Bootstrap 3.
    You can find that here: http://getbootstrap.com/getting-started
  3. Lastly you will need the plugin.
    Which can be found here: Download jpFileUploader
  4. Optional stuff.
    • Font Awesome
    • Toastr


Step 1.

Create a <DIV> element in your form with an ID of whatever you choose. Remember you can only have 1 instance of the jpFileUploader on a page.

Step 2.

Create the instance $("#yourDivId").jpfu(). Jump over to the options if you want some more control of your jpFileUploader

Step 3.

Profit!
